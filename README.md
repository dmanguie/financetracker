🎒 Student Finance Tracker
A modern, student-focused expense manager built with Kotlin and Material Design 3.

✨ Features
Neon Ledger UI: Dark mode aesthetic with high-contrast accents.

Budget Tracking: Set a monthly limit and see a real-time progress bar.

Categorization: Emoji-coded tags for Food, School, Transport, and more.

Month Filtering: View your spending habits month-by-month.
